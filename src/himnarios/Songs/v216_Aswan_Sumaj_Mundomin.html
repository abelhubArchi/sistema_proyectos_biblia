Aswan Sumaj Mundomin
# de Himno Verde #216
Presentación V1 C V2 C V3 C

Verso 1
Aswan sumaj mundomin tiyan,
K'anchaynimpi santos kawsanku.
Payllaman takinku kusiywan,
Jatun atiyniyoj Jesusman.

Coro
/Chay sumaj mundopi
Kawsasunchej Señor Jesuswan./

Verso 2
Payman takisunchej kusiywan
Angeleswan khuska cielopi.
Cruzpi yawarnintajich’arqa,
Juchasninchejta llimphucharqa.

Verso 3
Salvadorwan kawsamusunchej
Wiñaypaj chay k'anchaj mundopi.
Graciasta qosunchej sonqowan,
Yupaychaspa wiñay kusiypi.
