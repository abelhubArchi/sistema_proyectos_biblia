Señor Jesucristo Yanaparillaway
# de Himno Verde #96
Presentación V1 C V2 C V3 C

Verso 1
Señor Jesucristo, yanaparillaway
Qanllawan puriyta kay kawsayniypi.
Pisi kallpa kani, Qanman chimpamuyki;
Qan kallpachariway atiyniykiwan.

Coro
Señor Jesucristo, kallpacharillaway
Espíritu Santoj kallpachayninwan.
Uimphuchariwaytaj tukuy juchaymanta.
Qanman chimpamuyki tukuy sonqowan.

Verso 2
Ama saqewaychu juchaman urmajta,
Ruwayta munani munayniykita.
Qanta sirvispalla kusiyta tarini.
Juchata sirvejtíy llakiy lia kawan.

Verso 3
Pruebaspis, llakiypis jamullawampuni.
Atipasajpuni atiyniykiwan.
Kay pacha kawsaypi llakiyilaña junfa.
Qanllapiña tíyan wiñay kusiyqa.
