Jallp’a Chinkanqa
# de Himno Verde #113
Presentación V1 C V2 C V3 C V4 C

Verso 1
/Jallp'a chinkanqa, cielo chinkanqa,
Diospa Palabran ni chinkanqachu./

Coro
Diospa santo Ubron jinata nishan,
Chayrayku, hermano’ listo suyana.
Mana puñuspalla Payta suyana.

Verso 2
/Hermanitosníy, kallpachakuychej,
Diospa jamunan qayllapiñamin./

Verso 3
/Diosqa niwanchej wasis kasqanta
Creejkunapaj janaj pachapi./

Verso 4
/Hermanitosníy, hermanitasníy,
Diospa ladonman ripusunchejña./
