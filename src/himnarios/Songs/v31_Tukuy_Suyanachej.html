Tukuy Suyanachej
# de Himno Verde #31
Presentación V1 C V2 C V3 C V4 C V5 C

Verso 1
Tukuy suyanachej pisi tiempotawan
Ña kutírimonqaña Jesusninchej.
Oraspa, leespa, takispa qhawana.
Uyarisunchejña, "Kaypiña" nejta.

Coro
Señor Jesusniy ña jamonqaña,
Mañana Payqa tardanqañachu.

Verso 2
Arcangelpa sinch'i qhaparimuyninwan,
Trompetanwan wajyaspa Pay jamonqa.
Mayta suyashanchej Pay kutimunanta,
Puñojta kawsajta oqhariwasunchej.

Verso 3
Yachanchej Señorpi Puñojkunataqa
Señorwan khuskata Dios oqharenqa.
Paykunamantari mana ñawpasunchu,
Ujchasqasta tukuyta oqhariwasun.

Verso 4
Puñukapojkunaraj kawsarenqanku
Paykunata Cristo wajyajtinkama,
Kawsashajkunari ujchakullasuntaj.
Chanta Jesús phuyuspi oqhariwasun.

Verso 5
Jaqay cielopiri rikunakusunchej
Jesustari rejsisun uyamanta.
Paywantaj wiñaypaj kusikamusunchej
Cnstopi tukuy satvasqas kajkuna.
