Juchasneyqa Mayllasqa
# de Himno Verde #396
Presentación V1 C V2 C V3 C V4 C

Verso 1
Juchasneyqa mayllasqa Jesucristoj yawarninwan,
Sonqoypis llimphuchasqa Jesucristoj yawarniwan.

Coro
Yaku pujyujina rit’ita llimphuchan,
Manota] waj kanchu, Jesucristoj yawarnillan.

Verso 2
Juchaymanta junfarqa Jesucristoj yawarnillan,
Diosman qayllachiwarqa Jesucristoj yawarnillan.

Verso 3
Salvacionta tarini Jesucristoj yawarnimpi.
Thañikuytapis qowan Jesucristoj yawarnillan.

Verso 4
Kusikuywan takini Jesucristo Salvawajman.
Jesús, Diospa Corderon yawarninwan mayllawanchej.
