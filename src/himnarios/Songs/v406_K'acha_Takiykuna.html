K'acha Takiykuna
# de Himno Verde #406
Presentación V1 C V2 C V3 C

Verso 1
K'acha takiykuna Diosllapaj kachun,
Tukuynejpi uyarikushan.
Janaj pachamanta Cristo Salvador
Wañuymanta salvaj jamorqa.

Coro
Gloria, gloría kachun Dios llaman,
Tukuy yupaychana Payllata.
Chay Belén llajtapi yupaychaykuna
Uyarikullachuntaj kaypi.

Verso 2
Kusiywan junfasqa sonqonchej kachun.
Diosninchej khuyakun runasta.
Nak'ariy, nanaypi kashajkunaman
Dios rikuchin munakuyninta.

Verso 3
Sonqosninchejpitaj kusiy junfasqa,
Salvawajninchej Jesusmanta.
Yupaychana Payta takiykunawan,
Jesucristo Salvajninchejta.
