Janaj Pachamanta
# de Himno Verde #214
Presentación V1 C V2 C V3 C V4 C

Verso 1
Janaj pachamanta uyarikushan;
"K’anchayta apaychej.
Askha runas laqhayaypi kashanku,
K'anchayta apaychej."

Coro
Jesuspa sufi k'anchayninta
Apaychej tukuynejman.
Jesuspa cheqa k)anchayninta
Apaychej munakuywan.

Verso 2
Uyariychej Jesuspa nisqasninta,
K’anchayta apaychej.
Kasunachej Jesuspa nisqasninta,
K'anchayta apaspa.

Verso 3
Dios Tatáy, Qanlla yanapariwayku
K'anchayta apayta.
Kay pachapi tukuy runas rejsichun
Diospata k'anchayta.

Verso 4
Chay k'anchayta usqayta apanachej,
K'anchaypi kawsaspa,
Laqhayaymanta almasta orqhospa
Diospa k'anchayninman.
