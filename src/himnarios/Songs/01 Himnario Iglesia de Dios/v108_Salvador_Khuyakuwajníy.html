Salvador Khuyakuwajníy
# de Himno Verde #108
Presentación V1 C V2 C V3 C

Verso 1
Salvador khuyakuwajníy, chimpaykamuyki.
Qanlla qoyta atiwanki kusiy kawsayta.

Coro
Cristo, Cristo, Salvakuwajníy,
Uyaríyvay mañasqayta, khuyakoj Señor.

Verso 2
Salvacionniyta mask'aspa, chimpaykamuyki.
Noqapaj wañusqaykita creeni, Señor.

Verso 3
Khuyakuyniykiraykulla salvakuwanki.
Yawarniyki llimphucharqa ch'ichi sonqoyta.
