Huertaspi Rikunchej
T. Rafael
# de Himno Verde #376
Presentación V1 C V2 C V3 C

Verso 1
Huertaspi rikunchej durazno sach'asta,
Trigosta, sarasta, naranja sach'asta.
Chaykuna kashanku Diospa bendicionnin,
Diospa wawasnenqa agradecenachej.

Coro
Para kajtinkama, pampas q'omeryanku,
P'isqos takirinku, qhoraspis wiñanku.
Ch'uwa yakusjamun orqosmantapacha
Cristo, "kawsay yaku" tukuy ch’akisqaspaj.

Verso 2
Tataypis, mamaypis, tukuy wawqesniypis,
Tiempo kajtinkama saqepuwankuña.
Kay pacha kawsayqa tukukapojllamin.
Qhorajina wiñan, fikan ch^akipuntaj.

Verso 3
Munakoj Diosneyqa cielopi, jallp'api,
Tlikuynejpi kashan waqaychakushawan.
Tata Diosninchejta yupaychaykunachej
Qhuyakuyninmanta,bendidonninmanta.
