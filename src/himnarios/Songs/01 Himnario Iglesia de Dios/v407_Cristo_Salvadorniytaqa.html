Cristo Salvadorniytaqa
# de Himno Verde #407
Presentación V1 C V2 C V3 C

Verso 1
Cristo Salvawajniytaqa, Payllata munakusaj;
Chay tronompi tiyashajta rikumuyta munani.
Yachachejniy Jesusneyqa cruzpi ñak'arerqa Pay.
Kawsamusaj Payllawanña k'anchaypi, kusikuypi.

Coro
Rikumusaj chay lugarta, kusíkuyua chaypeqa kanqa,
Munani kawsakamuyta Jesusniy Salvadorwan.

Verso 2
Cristo Salvadorniytaqa, Payllata munakusaj;
Juchayojman willamusaj chay munakuyninmanta.
Cruzpi wañusqanmantapis, janaj pachamantapis.
Takimusaj angeleswan, Diosllata yupaychaspa.

Verso 3
Jesucristo Salvadorniy chay cruzpi wañusqanwan.
Kawsamusaj wiñaypajmin, munakuynillampiña.
Juchasniyta llimphucharqa Jesucristoj yawarnin,
Salvasqas kajkunawantaj takimusaj Señorman.
