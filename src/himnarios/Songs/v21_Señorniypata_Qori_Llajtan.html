Señorniypata Qori Llajtan
# de Himno Verde #21
Presentación V1 C V2 C V3 C

Verso 1
/Señorniypata qori llajtan
Wiñay wiñaypaj lliphirenqa./
/Dios Tatay chayta wakicherqa
Tukuynin wawasnillampajmin./

Coro
/Janaj pacha wasiyman risaj
Tukuy nanayía saqepusqa./

Verso 2
/Kay pachapeqa llakiy, nanay,
Janaj pachapi kusikuyila./
/Runa masiyki chejnisojtín,
Waqaychasonqa Dios Tatanchej./

Verso 3
/Mundo kusiyta mak'akuspa,
Llakiykunata tarikunchej./
/Janaj pachapi Dios Tatawan
Wiñay wiñaypaj kusikusun./
