Yachaychej Wasis
# de Himno Verde #217
Presentación V1 C V2 C V3 C

Verso 1
Yachaychej wasis kashasqanta
Diospa Iliphej qori llajtampi.
Creejkunaqa chayman renqanku
Jesuswan wiñaypaj kusikoj.

Coro
Aswanta cielopi
Yuyaychej wasis kashasqanta.
Aswanta jaqaypí cielopi,
Diospa Iliphej qori llajtampi.

Verso 2
Ñawpankuña chay hermanospis
Kusikushankuña cielopi.
Paykunawan rikunakusunchej
Diospa Iliphej qori llajtampi.

Verso 3
Yuyaychej chay k'anchaj llajtapi;
Chayman ayllusninchej rinkuña.
Noqanchejpis ña ripusunchejña,
Ni kanqachu nanay, waqaypis.
