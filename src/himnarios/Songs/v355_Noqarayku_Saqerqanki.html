Noqarayku Saqerqanki
# de Himno Verde #355
Presentación V1 C V2 C V3 C V4 C

Verso 1
Noqarayku saqerqanki tronoykita,
Belenpi nacej jamorqanki.
Wasipi mañana Qampaj campo kajtin,
Uywa corraipi nacerqanki.

Coro
/Jamuy ari. Jesús, sonqoyman,
Qampaj lugar wakisqa kashan./

Verso 2
Kusiy takiywan ángeles yupaychanku
Diospa Churin Jesucristota.
Wajcha kayman tukuspa, Qan jamorqanki,
Wiñay kawsayta qonaykipaj.

Verso 3
Atojkunajta tíyan puñunasninku
Tukuy phawajkunajpatapis.
Diospa Churimpaj mana karqachu lugar
Umanta sirichinampajpis.

Verso 4
Señorniy, jamorqanki runasta salvaj,
Kacharichej juchasninmanta.
Wañuchisorqanku Qanta, qhesachaspa
Jatun munakuyniykitapis.
