Getsemaní Lugarpeqa
# de Himno Verde #155
Presentación V1 C V2 C V3 C V4 C V5 C

Verso 1
Getsemaní lugarpeqa
Señor Jesús orasharqa:
"Munayniyki kachun, Tatáy,"
Nispa, orarqa Pay.

Coro
Ashkha runas chawpipitaj
Judas Iscariote risharqa.
Jesusta ríkuytawanrí,
Much'aj chimpaykorqa.

Verso 2
Runasqa chayta rikuspa,
Jesusta presocharqanku.
Judasrí, chay juchanmanta
Sipirpakamorqa.

Verso 3
Presota jap4spatajri,
Khishkaswan pillucharqanku.
Imayna nanaychus karqa
Señor Jesuspajqa.

Verso 4
Soldadostaj jasufispa,
Cruztaqa apacherqanku.
T'ojlu, nisqa orqopitaj
Chakataykorqanku.

Verso 5
Chay cruzpi wañuspatajrí,
Juchanchejta pampacharqa.
Chayraykutaj, hermanosníy,
Salvasqasña kanchej.
