Tukuy Uyarejqa
# de Himno Verde #417
Presentación V1 C V2 C V3 C

Verso 1
Tukuy uyarejqa willaramuchun,
Salvación kasqanta creesqanrayku.
Chinkasqa mundoman willamuy, nispa:
"Riy Salvador Jesusman."

Coro
Tukuy creejkuna mllaramuna,
Evangeliomanta sumaj mllayta.
Qhasilla qóy muñan perdonta Jesús.
"Riy Salvador Jesusman".

Verso 2
Salvakuyta munaj usqhayta richun,
Punku kichasoaña yaykúy munajpaj.
Cristollamin Nanqa janaj pachaman.
"Riy, Salvador Jesusman."

Verso 3
Nisqanta Jesusqa junfanqapuni,
Uyariy juchayoj, chimpay Señorman.
Payqa munakuywan tukuyta wajyan.
"Riy, Salvador Jesusman."
