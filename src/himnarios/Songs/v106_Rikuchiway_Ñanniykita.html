Rikuchiway Ñanniykita
G. Zeballos
# de Himno Verde #106
Presentación V1 C V2 C V3 C

Verso 1
Rikuchiway ñanniykita, munakoj Dios Tatáy.
Qanmanta mañakamuyki tukuy sonqoymanta.

Coro
Dios Tatáy, kf anchaban k'acha Palabrayki.
Ñanniykita rikuchiyvay Palabraykiwan, Dios.

Verso 2
Laqhayaypi k'anchaykuway qhatikunasuypaj.
Chay k'acha Palabrastaqa Señormin parlarqa.

Verso 3
Yanapaway kasukuyta kamachisniykita,
Almaytaj kallpachakonqa Palabrasniykiwan.
