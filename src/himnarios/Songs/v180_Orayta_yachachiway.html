Orayta yachachiway
# de Himno Verde #180
Presentación V1 C V2 C V3 C V4 C

Verso 1
Orayta yachachiway; Diosníy, Qanlla atinki.
Astawan feta qoway faltawasqanman jiña.

Coro
Dios Tatáy, uyaríway mañakususqaytaqa.
Allin kajilata qoway Jesusrayku. Amén.

Verso 2
Orayta yachachiway munayniykiman jiña.
Allinta kawsachiway Espirituykinejta.

Verso 3
Juchapi kani chayrí, Qan rikuchiway ari.
Juchayta willakuyki; sumaj sonqota qoway.

Verso 4
Graciapi wiñachiway, astawan munakuypi.
Qanta yupaychasqayki wiñay wiñaynintimpaj.
