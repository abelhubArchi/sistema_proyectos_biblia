Noqanchejwampuni
# de Himno Verde #314
Presentación V1 C V2 C V3 C V4 C

Verso 1
Noqanchejwampuni Dios kachun
Cieloman ripunakama.
Diospa ladompi kaspaña,
Wiñay wiñaypaj kusikusun.

Coro
Jesusninchej kutimojtinña,
Paywan khuska kusikusunchej.
Tantaykukusun mundomanta.
Michejninchej Diospa wawasnin.

Verso 2
Noqanchejwampuni Dios kachun.
Laqha tutas puriypipis,
Llakiy pacha kawsaypipis,
Paywan kaspa ni manchiy kanchu.

Verso 3
Noqanchejwampuni Dios kachun,
Llanthullampi pusawasun
Sinch'ita kallpachaykuspa
Sonqochakoj Espiritunwan.

Verso 4
Noqanchejwampuni Dios kachun.
Makisnimpi jap’wasun,
Lliphipej janaj pachaman
Pusakapunawanchejkama.
