Santa Bibliapi
# de Himno Verde #199
Presentación V1 C V2 C V3 C

Verso 1
Santa Bibliapi willawanchej Dios
Mayta munkuwasqanchejta Pay.
Mayllashaspa juchasninchejtaqa:
"Wawasniy kankichej," niwarqanchej.

Coro
donata qona Salvadorman,
Kusikuna, kusikuna.
Kusichiwanchej munakuynin,
Kusikuyninchej Pay.

Verso 2
Bibliapi Cristo runasman willan
Cruzman paykunarayku risqanta.
Jamun onqosqa runasta jampej,
Paypa jampinri cruzpi yawarnin.

Verso 3
"Sonqosniykichejta qowaychej," nin,
"Qhatiwaychejtaj cheqan ñanniypi."
Cruzpi wañuspa, rantiwarqanchej.
Kunan chimpaykuwajchej Payllaman.
