Jesusta Rikúy Munayman
# de Himno Verde #415
Presentación V1 C V2 C V3 C V4 C

Verso 1
Jesusta rikúy munayman. Imaynataj kanmanri?
Uj p'unchaytaj kusikuypi; almayqa qhawarenqa.

Coro
Rejsimusaj uyamanta Pay kikinta cielopi,
Rejsimusaj cielopiraj munakoj Jesusniyta.

Verso 2
Payta ni rikunirajchu, kay pacha ñawisniywan;
Jaqayjamoj p'unchaypiraj, Paytaqa rikurisaj.

Verso 3
Noqa chaypi kusikusaj, uyanta qhawarispa;
Tukuy imata saqespa, taripaj Payta risaj.

Verso 4
Máy sumajchus kawsay kanqa Paywanjanaj pachapi.
Almasninchejta salvajpa uyanta rejsimusaj.
