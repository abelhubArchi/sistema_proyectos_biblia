Santa Bibliaqa
# de Himno Verde #150
Presentación V1 C V2 C V3 C V4 C

Verso 1
Santa Bibliaqa Diospa Palabranmin.
Yachachiwanchej cheqa kajmanta,
Jesucristo Diospa chay k'ata Churín
Runasta salvaj jamusqanmanta.

Coro
/Leey Bibliata, kasuy Bibliata,
Uyariytaj Diospa nisqasninta./

Verso 2
Santa Bibliapi yachaychiykunaqa
Almasninchejman kawsayta qonqa,
Pichus leejqa k'anchaypi purenqa.
Wiñaypajtaj kusisqa kawsanqa.

Verso 3
Santa Bibliaqa yanapawasunchej
Tukuy pruebasta atípaytapis.
Janaj pachapi Jesusta rikuspa,
Paywan kusikusunchej wiñaypaj.

Verso 4
Graciasta qonachej Dios Tatanchejman;
Palabrannejta tukuyman parlan.
Kay pachapi tukuy kaj pasaponqa,
Diospata nisqasnin junfakonqa.
