Jamuy Ari Jesucristo
# de Himno Verde #324
Presentación V1 C V2 C V3 C

Verso 1
Jamuy ari, Jesucristo, jap'ikuway Diosníy, Qampaj
Wawayki chimpamuyki.
Tukuy sonqo tukuy, alma qayllamuyki, Señorníy,
Qampata kani, Diosníy.

Coro
Jamuy ari, Jesucristo, kay sonqoypi tiyakuy.
Qanllapaj kawsáy munani, almaywan sirvisqayki.

Verso 2
Qhechuway kay débil kayta, Ilakikuyta, sajra kayta,
Qhechuway, k'acha Jesús.
Qhechuway manchachikuyta, qhechuway, k'adia Jesús,
Qhechullaway, Jesusníy.

Verso 3
Qanllapaj jap’iy cuerpoyta, kawsayniyta, tukuyninta
Qanman qopuyki, Señor.
Noqarayku wañorqanki, chayrayku tukuy almayta
Jaywapuyki, Señorníy.
