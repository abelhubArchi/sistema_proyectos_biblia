Cristoj Jamunanqa
# de Himno Verde #218
Presentación V1 C V2 C V3 C V4 C

Verso 1
Cristoj jamunanqa junfakushanñamin.
Imanasunchejtaj?
Jatun munakuywan Payta suyasunchej
Jesús Salvawajninchejta.

Coro
Mañana sirvispa iskay dioskunata,
Cristo sapallanta tukuy sirvinachej.
Manamin Diostaqa, Supaytawan khuska
Sirvisunmanchu, hermanóy.

Verso 2
Iskayta sirvejqa mana kawsanqachu,
Wañuyilaman renqa;
Cristo sapallanta sirvejkunapajqa
Wiñay kawsay wakichisqa.

Verso 3
Imaymana nejta uyarishanchejña.
Qhawaychej allinta
Onqoykunatapis, chayrí ch'ajwastapis.
Qayllamunña juicio p'unchay.

Verso 4
Higos sach'amanta yuyarikunachej,
Q'omeryashajtinña,
Punkupiña kasaj, nerqa Jesusninchej,
Noqanchejwan kawsanqa Pay.
