Imaraykuchus Dios
# de Himno Verde #229
Presentación V1 C V2 C V3 C

Verso 1
Imaraykuchus Dios chay jinatapuni
Mundota munakorqa,
K'ata Churinta qorqa. Pillapis creej
Salvakunampaj.

Coro
/Ama mñaypaj pierdekunampaj,
Wiñay kawsayniyoj kanampaj./

Verso 2
Dios Churinta kay pachaman kachamorqa
Salvasqas kananchejpaj.
Creenachej Jesuswan kawsananchejpaj
Wiñay wiñaypaj.

Verso 3
Munakuyta rikuchiwanchej Diosninchej
Churinta kachamuspa,
Munakuywan noqanchejpis kawsanapaj
Kay pachapeqa.
