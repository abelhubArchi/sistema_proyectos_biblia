Diosninchej Apachimonqa
# de Himno Verde #394
Presentación V1 C V2 C V3 C V4 C

Verso 1
Diosninchej apachimonqa
Jatun bendicionninta.
Chayta mañakoj jamuyku.
Apachimuwayku, Dios.

Coro
Bendición esta parachimullay, Diosníy.
Chay munakuyniykimanta
Aswanta yapawayku.

Verso 2
Diosníy, bendicionesmanta
Sinch’i pisiypi kayku.
Chayta mañakoj jamuyku,
Kachamuwayku, Diosníy

Verso 3
Mañakoj chimpaykamuyku,
Uyariwayku, Diosníy.
Bendicionesniyki manta
Yupaychasqayku Qanta.

Verso 4
Parachimuwasqaykuña
Bendicionniykita, Dios.
Kaypi mayta suyashayku
Usqhayta jamunanta.
